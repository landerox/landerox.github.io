## [1.0.0] - 2025-12-08

Initial release of the personal documentation and portfolio site.

### Added

- Personal portfolio and blog built with Zensical.
- Initial site sections (Blog, Projects, Resume, Radar, Index).
- Custom domain configuration via CNAME file.

### Configuration / Chore

- Implemented pre-commit hooks for code quality (YAML, Markdown, etc.).
- Configured CI/CD pipeline (`deploy.yml`) for automated publishing to GitHub Pages.
- Added linting configuration for Markdown and YAML.
