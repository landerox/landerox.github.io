# :material-radar: Technology Stack Radar

> *A curated collection of repositories, high-performance tools, and emerging technologies I am currently exploring, benchmarking, and integrating into scalable workflows.*

## :material-database-cog: Data Engineering & High-Performance Computing

* **[Dagster](https://github.com/dagster-io/dagster)**
    An orchestration platform for the development, production, and observation of data assets, emphasizing developer experience.
* **[dbt-bigquery](https://github.com/dbt-labs/dbt-bigquery)**
    The adapter plugin allowing dbt to operationalize data transformation workflows directly within Google BigQuery.
* **[Polars](https://pola.rs/)**
    A lightning-fast DataFrame library implemented in Rust and Python, designed for effective parallelism and memory efficiency.
* **[DuckDB](https://github.com/duckdb/duckdb)**
    An in-process SQL OLAP database management system used for high-performance analytical queries on local data.
* **[Apache Iceberg](https://github.com/apache/iceberg)**
    A high-performance open table format for huge analytic datasets, enabling SQL behavior on data lakes.

## :material-robot: AI Engineering, RAG & LLM Ops

* **[LangChain](https://github.com/langchain-ai/langchain)**
    A framework for developing applications powered by language models using composable tools.
* **[LlamaIndex](https://github.com/run-llama/llama_index)**
    A data framework for LLMs to ingest, structure, and access private data, specifically optimized for RAG (Retrieval-Augmented Generation).
* **[DSPy](https://github.com/stanfordnlp/dspy)**
    The framework for programming—not prompting—language models, moving towards modular and optimized LM pipelines.
* **[Ollama](https://github.com/ollama/ollama)**
    Get up and running with large language models locally, essential for offline inference and privacy-focused development.
* **[Gemini CLI](https://github.com/google-gemini/gemini-cli)**
    Command-line interface for interacting with Google's Gemini models.
* **[Claude CLI](https://github.com/anthropic/claude-code)**
    Terminal interface for interacting with Anthropic's Claude models.
* **[GitHub Copilot CLI](https://githubnext.com/projects/copilot-cli/)**
    A CLI experience that translates natural language into shell commands using GitHub Copilot.

## :material-web: Modern Web Architecture & Documentation

* **[FastAPI](https://github.com/fastapi/fastapi)**
    A modern, fast web framework for building APIs with Python based on standard type hints.
* **[Pydantic](https://github.com/pydantic/pydantic)**
    Data validation using Python type hints; the backbone of modern Python data exchange.
* **[Streamlit](https://streamlit.io/)**
    An open-source Python library that transforms data scripts into shareable web apps in minutes.
* **[Zensical](https://github.com/zensical/zensical)**
    A modern static site generator (SSG) focused on performance and simplicity.
* **[MkDocs](https://www.mkdocs.org/)**
    A fast, simple static site generator that is geared towards building project documentation.

## :material-code-braces: Rust Ecosystem & Core Languages

* **[Rust](https://github.com/rust-lang/rust)**
    A systems programming language empowering everyone to build reliable and efficient software.
* **[Ruff](https://github.com/astral-sh/ruff)**
    An extremely fast Python linter and code formatter, written in Rust.
* **[UV](https://github.com/astral-sh/uv)**
    An extremely fast Python package installer and resolver, written in Rust.

## :material-console-line: Developer Environment & Tooling

* **[Cursor](https://www.cursor.com/)**
    The AI-first code editor designed to pair-program with you, built on VS Code.
* **[Google Antigravity](https://antigravity.google/)**
    A deep dive into Python's memory management and garbage collection visualization.
* **[Tabby](https://github.com/Eugeny/tabby)**
    A highly configurable terminal emulator for the modern age.
* **[Oh My Zsh](https://ohmyz.sh/)**
    A framework for managing your Zsh configuration.
* **[Starship](https://starship.rs/)**
    The minimal, blazing-fast, and infinitely customizable prompt for any shell.
* **[Git](https://git-scm.com/)**
    The standard for distributed version control.
