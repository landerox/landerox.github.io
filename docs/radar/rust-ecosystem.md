# :material-language-rust: Rust Ecosystem

> Modern CLI tools, task runners, and web frameworks written in Rust.

## CLI Tools

| Tool | Replaces | Why |
|------|----------|-----|
| [**ripgrep (rg)**](https://github.com/BurntSushi/ripgrep){ target="_blank" } | grep | Ultra-fast recursive search, respects .gitignore |
| [**fd**](https://github.com/sharkdp/fd){ target="_blank" } | find | Simple syntax, smart defaults, gitignore-aware |
| [**bat**](https://github.com/sharkdp/bat){ target="_blank" } | cat | Syntax highlighting, line numbers, Git integration |
| [**eza**](https://github.com/eza-community/eza){ target="_blank" } | ls | Colors, icons, Git status (actively maintained exa successor) |
| [**zoxide**](https://github.com/ajeetdsouza/zoxide){ target="_blank" } | cd | Smart autojump—learns your most-used directories |
| [**dust**](https://github.com/bootandy/dust){ target="_blank" } | du | Intuitive disk usage visualization |
| [**delta**](https://github.com/dandavison/delta){ target="_blank" } | diff | Beautiful Git diffs with syntax highlighting |
| [**just**](https://github.com/casey/just){ target="_blank" } | make | Modern task runner—simple syntax, no .PHONY nonsense |
| [**hyperfine**](https://github.com/sharkdp/hyperfine){ target="_blank" } | time | Statistical benchmarking with warmup runs and comparisons |
| [**starship**](https://starship.rs/){ target="_blank" } | prompt | Blazing-fast, highly customizable cross-shell prompt |

## Web Frameworks & Runtime

| Resource | Description |
|----------|-------------|
| [**Axum**](https://github.com/tokio-rs/axum){ target="_blank" } | Ergonomic web framework from the Tokio team—fastest growing in adoption |
| [**Actix Web**](https://actix.rs/){ target="_blank" } | Maximum performance—consistently tops web framework benchmarks |
| [**Loco**](https://loco.rs/){ target="_blank" } | Rails-like framework—batteries included for rapid development |
| [**Tokio**](https://tokio.rs/){ target="_blank" } | Async runtime foundation—powers most of the Rust async ecosystem |
